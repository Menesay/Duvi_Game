## Overview

DUVÄ° GAME is a simple yet engaging game built using Python's Tkinter library. The objective is to control the character (Vaddab) and avoid colliding with the moving enemies (Duvi's) on the screen. The game includes a stopwatch to track the player's survival time and a menu interface for navigation.

## Installation

1. Ensure you have Python installed on your machine.
2. Install the Tkinter library if it's not already included:
   ```sh
   pip install tk
   ```
3. Download the game files, including the images `duvi.png`, `Duvi_Game.png`, and `vaddab.png`.

## Controls

- **Move Up:** Press `W`
- **Move Down:** Press `S`
- **Move Left:** Press `A`
- **Move Right:** Press `D`
- **Play Again:** Press `R` (when game over)
- **Quit:** Press `Q` (when game over)
- **Back to Main Menu:** Click the "Back" button
